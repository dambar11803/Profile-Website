INTRODUCTION
PATH TRACKER ROBOT is a simple robot. This is a simple robot that was designed as a Third Year Mini Project just to know the basics of some electronics components and sensors. The main aim of this project is to be familiar with the basic robotic circuit which can help to advance the major projet in the Main Project. Here, a light sensor is used to track the white path and to move the mechanical body alng the tracked path.

FUNCTIONS

  • Just walks following the defined path.

ADVANTAGES

  • Easy to use
  • User-Friendly
  • Understanding basics and be familirar with the robotic circuit, electronic components and sensors.

APPLICATIONS

  • Industrial Purposes
  • Commercial Purposes
  • Research and Development
