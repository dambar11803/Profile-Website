INTRODUCTION
RATIO ANALYSIS SYSTEM is a Django based web application. This application helps for ratio analysis during credit assessment and credit rating. It is an important part of credit lending process because it helps to assess and analyze various kinds of financial ratios which further assist to refine in decision making during loan approval.

FUNCTIONS

  • Helps in Ratio Analysing
  • Assist in CFR Generation

ADVANTAGES

  • Easy to use
  • User-Friendly

APPLICATIONS

  • Banks & Finanacial Institutions
