INTRODUCTION
WI-FI CONTROLLED ROBOT is a ARDUINO based mechanical robot. The controlling of robot is based on wireless signal coming via wireless device like laptop, wireless remot etc. The robot can be controlled from any place in the world where network access is available.

FUNCTIONS

  • It can be used to take photos from remote areas.
  • It can be made advanced for other applications

ADVANTAGES

  • Easy to use
  • User-Friendly
  • It can be controlled via portable device like remote, laptop, mobile etc.

APPLICATIONS

  • Industrial Purposes
  • Commercial Purposes
  • Research and Development
